Question-1: What are the different ways to select an element in the DOM?

Use `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, or `querySelectorAll` to select elements.

getElementById: Selects an element by its ID.
getElementsByClassName: Selects elements by their class name.
getElementsByTagName: Selects elements by their tag name.
querySelector: Selects the first element that matches a specified CSS selector(s) in the document.
querySelectorAll: Selects all elements that match a specified CSS selector(s) in the document.
For Instance: 

document.getElementById('myId');

document.getElementsByClassName('myClass');

document.getElementsByTagName('div');

document.querySelector('#myId');
document.querySelector('.myClass');
document.querySelector('div');

document.querySelectorAll('.myClass');
document.querySelectorAll('div');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

`innerHTML` includes HTML tags, `innerText` returns visible text, and `textContent` returns all text, including hidden.

innerHTML: Sets or returns the HTML content (inner HTML) of an element.
innerText: Sets or returns the text content of the specified node, and all its descendants.
textContent: Sets or returns the textual content of a node and all its descendants. 

document.getElementById('myId').innerHTML;
document.getElementById('myId').innerText;
document.getElementById('myId').textContent;

Question-3: What is event delegation in the DOM?

Attach one event listener to a parent element to handle events for its children. Use `event.target` to identify the clicked element.
For Instance: 

document.getElementById('parent').addEventListener('click', function(event) {
  if (event.target.tagName === 'li') {
    console.log('List item clicked!');
  }
});

Question-4: What is event bubbling in the DOM?

An event starts from the element that triggered it (the target) and then "bubbles up" through its ancestors in the DOM tree.For example, if you click on a button inside a div, the click event doesnâ€™t just stay on the button. It bubbles up to the div, then to its parent, and so on, all the way up to the document object (unless someone stops it along the way). For Instance: 

document.getElementById('parent').addEventListener('click', function(event) {
  console.log('Clicked on parent!');
});
document.getElementById('child').addEventListener('click', function(event) {
  console.log('Clicked on child!');
});

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element, use `document.createElement()`. Pass the tag name (like `div` or `p`) to create the element.
For Instance: 

const newDiv = document.createElement('div');

To add the element to the DOM, use `appendChild()` or `insertBefore()` method.
For Instance: 
document.body.appendChild(newDiv);
document.body.insertBefore(newDiv, document.body.firstChild);

To remove the element from the DOM, use `removeChild()` method.
For Instance: 
document.body.removeChild(newDiv);
